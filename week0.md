# What did i learn this weeks?
I studied vectors from applied linear algebra. Read topics on linear and affine functions. 
Linear function 
``` f(x) = a^{T}x ```
f is linear iff ``` f(\alpha x + \beta y) = \alpha f(x) + \beta f(y)```
- Affine function 
``` f(x) = a^{T}x + b ```
f is affine iff ``` f(\alpha x + \beta y) = \alpha f(x) + \beta f(y)```
as long as ```\alpha + \beta = 1```
